What is it?

A simple tool for drawing geometric shapes in your browser. Started with a desktop app for learning geometry, ended up as a web app because why not.

You can sketch shapes, get basic info about them (area, perimeter), and transform them. Nothing complicated, just functional.

What You Can Do

Draw Shapes

Triangles, squares, rectangles, circles, or freeform polygons. Click points on the canvas and hit finish.

Quick Math

Automatic calculations for perimeter and area. Uses the Shoelace formula for polygons.

Transform

Rotate, scale, mirror horizontally or vertically. All transformations are instant.

Grid & Snap

Optional grid to help you draw straight. Can turn it off if you prefer freeform.

Save Work

Download your drawings as JSON. Load them back later if needed.

Keyboard Control

F to finish, Z to undo, Q to clear. Standard shortcuts for fast workflow.

Getting Started

Quick Start

git clone https://github.com/zhirkoalexander-maker/bender.git
cd bender/web

pip install -r requirements.txt
python app.py

Then go to http://localhost:5000

Requirements

  • Python 3.7+
  • Flask (installed via requirements.txt)

What you get

  • Web interface with HTML5 Canvas
  • Flask API backend for calculations
  • Real-time shape analysis
  • Transform tools (rotate, scale, mirror)
  • Grid system with snap-to-grid

How to Use It

  1. Pick a shape from the left panel
  2. Click on the canvas to place points
  3. Hit F, double-click, or click Finish button to complete
  4. Use the transform tools if you want to modify it
  5. Click Save to download as JSON

Basic Workflow

  1. Select Triangle (or whatever shape)
  2. Click three points on the canvas
  3. Press F or double-click to finish
  4. Area and perimeter show up at the bottom
  5. You can rotate it, scale it, mirror it if needed

Keyboard Shortcuts

F Finish current shape
Z Undo last action
Q Clear everything

API Reference

If you're running the Flask version locally, these endpoints are available:

Endpoint Method What it does
/api/analyze POST Get area, perimeter, shape type
/api/rotate POST Rotate shape by angle
/api/scale POST Scale shape by factor
/api/mirror POST Mirror horizontally or vertically
